微信小程序模板与配置 #

1.数据绑定 #

数据绑定基本原则

  • 在 data 中定义数据
  • 在 WXML 中使用数据

1.在 data 中定义页面的数据 #

在页面对应的.js文件中，把数据定义到 data 对象中即可：

Page({
    data:{
        // 字符串类型的数据
        info:'init data',
        // 数组类型的数据
        msgList:[{msg:"hello"}, {msg:'world'}]
    }
})

2.Mustache 语法的格式 #

把data中的数据绑定到页面中渲染，使用 Mustache 语法（双大括号）将变量包起来即可。语法格式为：

<view> {{ 要绑定的数据名称 }} </view>
1

3.Mustache 语法的应用场景 #

Mustache 语法的主要应用场景如下：

  • 绑定内容
  • 绑定属性
  • 运算（三元运算、算术运算等）
